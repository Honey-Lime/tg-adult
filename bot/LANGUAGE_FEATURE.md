# Функционал выбора языка / Language Selection Feature

## Описание

Добавлена возможность выбора языка интерфейса (Русский/English) с сохранением в базе данных.

## Изменения

### 1. База данных (`database.py`)

- Добавлен столбец `language` в таблицу `users` (по умолчанию 'ru')
- Функция `get_user_language(user_id)` - получение языка пользователя
- Функция `set_user_language(user_id, language)` - установка языка пользователя
- Обновлена функция `get_or_create_user()` - добавлен параметр `language`
- Обновлена функция `update_user_profile()` - добавлен параметр `language_code`

### 2. Клавиатуры (`keyboards.py`)

- Обновлена `get_main_menu_keyboard()` - добавлена кнопка "🌐 Language / Язык"
- Добавлена `get_language_keyboard()` - клавиатура выбора языка с флагами 🇷🇺/🇺🇸

### 3. Бот (`bot.py`)

- Обновлён `cmd_start()` - определение языка из `message.from_user.language_code`
- Обновлён `_update_user_profile_from_message()` - сохранение языка из сообщения
- Обновлён `_update_user_profile_from_callback()` - сохранение языка из колбэка
- Добавлены обработчики:
  - `callback.data == "language"` - показ меню выбора языка
  - `callback.data == "lang_ru"` - установка русского языка
  - `callback.data == "lang_en"` - установка английского языка

### 4. Автоматическая миграция

- Функция `init_db()` автоматически проверяет наличие столбца `language` и добавляет его при запуске бота

## Как это работает

1. **При первой регистрации:**
   - Бот определяет язык пользователя из `message.from_user.language_code`
   - Если язык начинается с 'ru' → устанавливается 'ru', иначе 'en'
   - Язык сохраняется в базе данных при создании пользователя

2. **При каждом взаимодействии:**
   - Язык пользователя обновляется из данных Telegram (автоматически)

3. **Ручное изменение:**
   - Пользователь нажимает кнопку "🌐 Language / Язык" в главном меню
   - Выбирает один из двух языков: 🇷🇺 Русский или 🇺🇸 English
   - Выбранный язык сохраняется в базе данных

## Использование

### Для пользователей:

- Нажмите кнопку "🌐 Language / Язык" в главном меню
- Выберите нужный язык

### Для разработчиков:

```python
# Получить язык пользователя
lang = database.get_user_language(user_id)

# Установить язык
database.set_user_language(user_id, 'ru')  # или 'en'
```

## Миграция

Миграция выполняется автоматически при запуске бота. Функция `init_db()` проверяет наличие столбца `language` и добавляет его, если он отсутствует. Все существующие пользователи получат язык 'ru' по умолчанию.
