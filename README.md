# Adult Telegram Bot

#

Telegram-бот для просмотра и оценки изображений (аниме/реальные фото) и видео.

## Основные функции

### Для пользователей:

- 🖼 Просмотр изображений двух типов (аниме и фото)
- 🎞 Просмотр видео (ТОП25, Хорошее, Без оценки)
- ❤️/😐 Лайки и дизлайки с начислением монет
- 💾 Сохранение изображений в коллекцию (за 25 монет)
- 🔗 Реферальная система (250 монет за приглашённого друга)
- 📊 Mini-приложение с топом и сохранёнными изображениями

### Для администраторов:

- 👥 Статистика пользователей
- 🛡 Модерация изображений
- 📢 Рассылка оповещений
- 🔗 **Рекламные ссылки** - создание и отслеживание статистики переходов
- 📥 Загрузка контента из JSON
- 🧹 Чистка по JSON
- 📋 Просмотр логов

## Рекламные ссылки

### Создание:

1. В админ-панели выберите "Рекламные ссылки"
2. Нажмите "Создать ссылку"
3. Введите название ссылки (первое сообщение станет названием)
4. Бот сгенерирует уникальную ссылку вида: `https://t.me/botname?start=CODE`

### Статистика:

- Кнопка "Статистика" показывает все созданные ссылки
- Для каждой ссылки отображается:
  - 📛 Название
  - 👥 Количество переходов
  - 🔗 Сама ссылка

### Удаление:

- Кнопка "Удалить ссылку" показывает пронумерованный список всех ссылок
- Отправьте номер ссылки для её удаления
- После удаления бот возвращает меню рекламных ссылок

### Как работает:

- При переходе по ссылке новый пользователь автоматически учитывается в статистике
- Записывается время перехода
- Переходы уникальных пользователей считаются один раз

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r bot/requirements.txt`
3. Настройте `.env` файл с параметрами:
   - `BOT_TOKEN` - токен бота
   - `DB_NAME`, `DB_USER`, `DB_PASSWORD`, `DB_HOST`, `DB_PORT` - параметры PostgreSQL
   - `ADMIN_IDS` - ID администраторов через запятую
4. Запустите бота: `python bot/bot.py`

При запуске бот автоматически проверит и добавит необходимые столбцы в базу данных.

## Загрузка нового контента

Актуальная загрузка контента работает через админку и файл:

- `bot/images/new/import.json`

### Куда класть файлы

Перед загрузкой новые файлы нужно положить в одну из папок:

- `bot/images/new/anime/` — новые anime-фото
- `bot/images/new/real/` — новые real-фото
- `bot/images/new/videos/` — новые видео

После успешного импорта бот перемещает файлы в рабочие папки:

- `bot/images/anime/`
- `bot/images/real/`
- `bot/images/videos/`

### Формат `import.json`

Файл должен быть словарём по датам. Для каждой даты указываются списки `pictures` и `videos`.

Пример:

```json
{
  "2026-05-22": {
    "pictures": [
      "anime/photo1.jpg",
      "anime/photo2.jpg"
    ],
    "videos": [
      "videos/video1.mp4"
    ]
  },
  "2026-05-23": {
    "pictures": [
      "real/photo3.jpg"
    ],
    "videos": []
  }
}
```

### Важные правила

- для фото путь должен начинаться с `anime/` или `real/`
- для видео обычно используется `videos/...`
- в одной записи даты не стоит смешивать `anime` и `real` фото
- если имя файла уже существует в БД, такой файл будет пропущен как дубль
- пропущенные дубли не считаются ошибкой и показываются отдельно в отчёте

### Как загрузить

1. Подготовьте файлы в `bot/images/new/...`
2. Заполните `bot/images/new/import.json`
3. В админ-панели нажмите **"Загрузка контента"**
4. Дождитесь отчёта по:
   - добавленным фото
   - добавленным видео
   - пропущенным повторяющимся файлам
   - ошибкам

### Очистка папки импорта

В админке есть отдельная команда очистки папки загрузки.

Она полностью удаляет содержимое `bot/images/new/`, включая:

- подпапки `anime`, `real`, `videos`
- `import.json`

Используйте её только после проверки, что импорт завершился корректно.
